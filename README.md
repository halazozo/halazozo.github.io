# Hala Zouzou Portfolio Website

A colorful, readable one-page portfolio website.

## How to publish
Upload the whole `hala_portfolio_site` folder to Netlify, GitHub Pages, or your hosting service.

Important: keep the `certificates` folder next to `index.html`, otherwise the certificate PDF buttons will not open.

## Privacy
This site does not include EID details, license numbers, phone number, passport number, home address, or diploma scans.
It only mentions Golden Visa holder and UAE driving license holder without private document details.
